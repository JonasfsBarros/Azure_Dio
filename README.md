# Repositório de Estudos sobre Azure

Este repositório tem como objetivo reunir resumos, anotações e dicas práticas sobre o uso da plataforma Microsoft Azure, servindo como material de apoio para estudos e futuras implementações.

📘 Resumos

1. Conceitos Fundamentais

Azure Resource Manager (ARM): camada de gerenciamento que organiza recursos em grupos.

Regiões e Zonas de Disponibilidade: escolha estratégica para latência e redundância.

Modelos de Serviço: IaaS, PaaS e SaaS.

2. Principais Serviços

Compute: Máquinas Virtuais, App Services, Functions.

Storage: Blob, File, Queue, Table.

Networking: Virtual Network, Load Balancer, VPN Gateway.

Banco de Dados: Azure SQL Database, Cosmos DB.

3. Segurança e Governança

Azure Active Directory (AAD): gerenciamento de identidade.

Role-Based Access Control (RBAC): controle de permissões.

Azure Policy: aplicação de regras e conformidade.

📝 Anotações

Escalabilidade: sempre considerar autoscaling para aplicações críticas.

Monitoramento: usar Azure Monitor e Application Insights.

Custos: acompanhar com Azure Cost Management.

Infraestrutura como Código: preferir ARM Templates ou Terraform.

💡 Dicas Práticas

Organização de Recursos: use Resource Groups para separar ambientes (dev, test, prod).

Nomenclatura: adote padrões claros para facilitar manutenção.

Backup e Recuperação: configure Azure Backup e Site Recovery.

Segurança: habilite MFA no Azure AD e use Key Vault para segredos.

Documentação: mantenha um README atualizado com instruções de uso.

✅ Objetivo

Este repositório deve servir como guia rápido e material de referência para estudantes, profissionais e equipes que desejam implementar soluções na nuvem utilizando o Microsoft Azure.
